# HR Strategic Dashboard – Power BI

**Data-Driven HR Insights • Modern • Professional**

An interactive and ergonomic Power BI dashboard that transforms raw HR data (from Excel) into actionable strategic insights, optimizing talent management, retention, and organizational performance.


## Project Overview

Designed for HR directors, managers, and decision-makers, this dashboard provides a clear and visual analysis of key HR metrics, facilitating data-driven decisions.

- Intuitive navigation via fixed left-side menu  
- Dynamic filters (Date, Department, Gender) synchronized across pages  
- Two complementary pages for a global and detailed view  
- Corporate design, responsive, and ready for presentations  

The purpose of this project is to convert raw HR data into a visual tool that identifies trends (such as turnover or promotions), analyzes employee demographics, and supports concrete HR strategies to improve retention, diversity, and overall performance.

## Dashboard Pages

### 1. Strategic Insights  
![Strategic Insights Page](./Accueil%20RH.png)
This page offers an executive overview of HR dynamics:  
- Turnover rates by age and hiring cohorts (curves and trends)  
- Employee performance with rankings  
- Promotion distribution (promoted vs. non-promoted)  
- Active vs. exiting staff comparison  
- Analysis of departure reasons to identify improvement levers  

Ideal for strategic reviews and quick trend identification.

### 2. Key Data & Demographics  
![Key Data Page](./Donn%C3%A9es%20Cl%C3%A9s.png)
An in-depth demographic analysis of the workforce:  
- Distribution by gender, city, education level, and age  
- Age pyramid by gender to visualize diversity  
- Average service duration and departure reasons  
- Headcount mapping by location  
- Insights into recruitment and retention  

Perfect for HR audits and diversity/inclusion plans.

## Key Features

- Fixed menu for smooth navigation between pages  
- KPI cards at the top for immediate access to essential metrics  
- Full interactivity: cross-filtering and drill-down  
- Modern visuals, harmonious colors, zero clutter  
- Mobile-responsive for on-the-go access  

## Technologies & Skills

- **Microsoft Power BI Desktop**: Full report design  
- **Power Query**: Data cleaning, transformation, and modeling from Excel  
- **DAX**: Advanced measures (turnover rates, performance scores, dynamic aggregations)  
- **UX/UI Best Practices**: Consistent colors
